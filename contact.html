FAQ

Before you send a request

Quick answers to help homeowners understand how PipeMatch works before comparing local plumbing provider options.

Start request
01 Do I need to know the exact plumbing problem?

No. You can start with symptoms. Choose the closest category and add a short description.

02 Can I request a planned plumbing upgrade?

Yes. Fixture installation, water heater replacement, and other planned upgrades can be submitted too.

03 Why do you ask for ZIP code and timing?

Location and urgency help match the request with more relevant provider options.

04 Does PipeMatch guarantee contractor work?

No. PipeMatch is an independent matching platform. Homeowners should verify licensing, insurance, and suitability before hiring.
