FAQ

Common questions before sending a request

Do I need to know the exact plumbing problem before submitting a request?

No. Many homeowners begin with symptoms rather than a full diagnosis. Choosing the closest service category and adding a short description is usually enough to start.

Can I submit a request for a planned plumbing upgrade?

Yes. Not every request is urgent. Planned upgrades such as fixture installation or water heater replacement can also be submitted through the same request flow.

Why does the form ask for ZIP code and timing?

Location and urgency can affect which provider options are most relevant to the request, so these details help frame the comparison more clearly.

Does PipeMatch guarantee any contractor’s work?

No. PipeMatch is an independent matching platform. Homeowners should independently verify licensing, insurance, and suitability before hiring any contractor.
